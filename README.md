# WQLog
Console Log
<img src="https://github.com/AppleDP/WQLog/blob/master/Effect/Effect.png" alt="Console Log" title="Console Log">
# Install 
```objective-c
pod 'WQLog', '~> 1.0.0'
```
# Usage
```objective-c
    // 普通日志输出
    WQLog(@"log");
    
    // 带色日志输出
    WQLogInfo(@"Log");
    WQLogError(@"Log");
    WQLogWarn(@"Log");
    WQLogMsg(@"Log");
```
